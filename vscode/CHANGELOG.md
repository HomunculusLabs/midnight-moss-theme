# Change Log

All notable changes to the "Midnight Moss Theme" extension will be documented in this file.

## [1.0.0] - 2024-10-22

### Initial Release

- Complete dark theme with moss-inspired green-teal accents
- Comprehensive editor color customization
- Full syntax highlighting for all major languages
- Terminal color scheme integration
- Optimized contrast ratios for extended coding sessions

### Theme Features

- **Primary accent**: Moss green-teal (#5e8d87) for selections, links, and highlights
- **Syntax colors**: Carefully chosen palette for excellent code readability
  - Keywords: Sage purple (#826d94)
  - Functions: Ocean blue (#5d7e97)
  - Strings: Amber (#a09262)
  - Classes: Forest green (#6e8f7c)
  - Comments: Muted gray (#7a7a7a)
- **UI elements**: Consistent styling across all VS Code panels
- **Terminal integration**: Full ANSI color palette support
