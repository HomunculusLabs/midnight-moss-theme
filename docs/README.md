# Midnight Moss - GitHub Pages Site

This directory contains the source files for the Midnight Moss theme landing page, hosted via GitHub Pages.

## Files

- `index.html` - Main landing page
- `style.css` - Stylesheet using Midnight Moss color palette
- `script.js` - Interactive features (color copying, scroll animations)
- `_config.yml` - Jekyll configuration

## Local Development

To preview the site locally:

1. Install Jekyll:
   ```bash
   gem install bundler jekyll
   ```

2. Serve the site:
   ```bash
   cd docs
   jekyll serve
   ```

3. Open http://localhost:4000/midnight-moss-theme in your browser

## Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the main branch.

Visit the live site at: https://homunculuslabs.github.io/midnight-moss-theme

## Features

- Interactive color palette (click to copy hex codes)
- Responsive design
- Smooth scroll animations
- Code preview with syntax highlighting
- Starfield background effect
